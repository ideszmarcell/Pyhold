PyHold - Pygame Játék
====================

Telepítés:
  1. Python 3.x szükséges (https://python.org)
  2. Telepítsd a Pygame-et: pip install pygame
  3. Futtasd: python main.py

Irányítás:
  - WASD vagy Nyilak: Mozgás
  - SPACE: Lövés (a mozgás irányába)
  - ESC: Kilépés
  - R: Újrakezdés (Game Over után)

Fájlok:
  - main.py          Belépési pont
  - game.py          Fő játék osztály
  - settings.py      Beállítások, konstansok
  - vector2.py       2D vektor osztály
  - entity.py        Alap entitás
  - player.py        Játékos
  - bullet.py        Lövedék
  - enemy.py         Ellenségek
  - pickup.py        Felszedethető tárgyak
  - hud.py           Kijelző (HUD)
  - wave_manager.py  Hullámkezelő
